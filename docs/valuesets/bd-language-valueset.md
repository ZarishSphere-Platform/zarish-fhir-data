# Allowed Languages

Only English and Bengali are allowed

## Overview

- **ID**: `bd-language-valueset`
- **Name**: BDLanguageVS
- **URL**: `https://fhir.dghs.gov.bd/core/ValueSet/bd-language-valueset`
- **Status**: draft

## Composition

This value set includes codes from the following code systems:

- `urn:ietf:bcp:47`

```json

  }]
}
```
